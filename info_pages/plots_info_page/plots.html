Problem generator

Примеры

Графики функций

Verjbei|Alfa может визуализировать математические функции.

Функции

Линейные графики

y=x
y=3x+10
y=3+10(x+3)

Уравнения со степенями

Пример правильного ввода

y=x^2+x^3-x^4-34
y=sqrt(x)+x^2

Уравнения с тригонометрическими функциями

Пример правильного ввода

y=sin(x)+cos(x)
y=tan(x)^2
y=cot(x)-3*sin(x)

Несколько уравнений

Пример правильного ввода

y=sin(x), cos(x),x^2
y=x^3-3;x+2;x^2-3x-1
